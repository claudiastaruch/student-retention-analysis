# student-retention-analysis
Analysis of student retention and performance trends using R

## Overview
This project analyzes student retention and performance trends across multiple academic programs, including ESP/RSP-supported courses.

## Data
- Aggregated student-level data
- Course enrollments, grades, and term outcomes
- Personally identifiable information removed

## Tools Used
- R (dplyr, ggplot2)

## Key Questions
- How do retention rates differ across programs?
- What trends exist over time?

## Results
- Identified consistent retention gaps across programs
- Produced summary tables and visualizations for reporting
